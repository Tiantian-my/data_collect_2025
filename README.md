# 哈尔滨城市冰雪数据集采集工具
硬件平台：采集计算设备（1-2台）、环视摄像头* 6、激光雷达* 1、GPS模块、IMU模块、框架和连接件、标定板、采集车辆等。

采集乘员：驾驶员*1、采集监控员 *2
## 计算设备分工
### 设备0
- 负责采集cam_0~2 + LiDAR数据
- 作为chrony时间同步主设备
### 设备1
- 负责采集cam_3~5 + IMU + GPS数据
- 作为chrony时间同步从设备
## 采集流程
0. 安全性检查
1. 采集平台固定和设备连接
2. 设备启动调试和标定数据采集
3. 出车和城市数据采集
4. 设备回收清点
## 后续工作
5. 组帧和必要的后处理
6. 数据标定和标注
